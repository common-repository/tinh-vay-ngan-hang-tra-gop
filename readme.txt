"Tinh Lai Vay Ngan Hang Tra Gop" is a plugin to calculate repayment schedule for individuals who borrow at credit institutions in Vietnam.

== Description ==

"Tinh Lai Vay Ngan Hang Tra Gop" is a plugin to calculate repayment schedule for individuals who borrow at credit institutions in Vietnam.

The plugin has 2 feature functions. These are the repayment schedule based on the original loan balance (Addon) and the repayment schedule according to the descending balance.

The pro version adds the function to send the repayment schedule to customers's email. You can check and buy at https://moneynextdoor.com

You can also check out the [live demo](https://moneynextdoor.com/tinh-lai-vay/) for details.

Documents link here: <a href="https://moneynextdoor.com/huong-dan-su-dung-plugin-tinh-lai-vay-ngan-hang-tra-gop/">https://moneynextdoor.com/huong-dan-su-dung-plugin-tinh-lai-vay-ngan-hang-tra-gop/</a>

== Installation ==

From within WordPress

1. Visit **Plugins > Add New**
2. Search for **Tinh Lai Vay Ngan Hang Tra Gop**
3. Click the **Install Now** button to install the plugin
4. Click the **Activate** button to activate the plugin
5. Add shortcode [tlvtg] in anny page or post you want to show plugin.
